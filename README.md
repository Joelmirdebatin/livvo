# LIVVO — Sistema de Gestão

Dashboard completo para gestão de clientes, financeiro, projetos e tarefas.

## Sobre

Aplicativo web standalone (HTML + React empacotado em arquivo único). Não requer servidor, banco de dados ou processo de build. Basta servir o `index.html` como conteúdo estático.

## Seções

- **Dashboard** — visão geral com indicadores
- **Clientes** — gestão da base de clientes
- **Financeiro** — controle de receitas e despesas
- **Projetos** — acompanhamento de projetos ativos
- **Tarefas** — organização de tarefas e entregas

## Publicação

Este projeto está hospedado no GitHub Pages. Qualquer alteração no arquivo `index.html` é publicada automaticamente na próxima sincronização.

## Stack

- HTML5
- React (empacotado inline)
- CSS inline (sem dependências externas de build)
